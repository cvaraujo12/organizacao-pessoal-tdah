# Sistema de Sons

Este sistema foi desenvolvido para reforçar positivamente suas conquistas através de feedback sonoro, criando um ciclo de recompensa mais efetivo para o cérebro TDAH.

## Tipos de Sons

### Sons de Conquista
- **Pequenas Conquistas**: Som curto e suave (0.5s)
- **Médias Conquistas**: Som médio e animador (1s)
- **Grandes Conquistas**: Som longo e celebrativo (1.5s)

### Sons de Streak
- **3 Dias**: Som de progresso inicial
- **7 Dias**: Som de progresso consistente
- **30 Dias**: Som de grande conquista

## Configurações

### Volume
- Volume padrão: 70%
- Ajustável de 0% a 100%
- Configurável por tipo de som

### Horário Silencioso
- Padrão: 22:00 - 08:00
- Personalizável via comando

## Comandos Úteis

- `/som_status` - Ver configurações atuais
- `/som_volume [0-100]` - Ajustar volume
- `/som_toggle` - Ativar/desativar sons
- `/som_silencioso [inicio] [fim]` - Definir horário silencioso

## Dicas para TDAH

1. **Mantenha o Volume Adequado**
   - Som alto demais pode ser distrator
   - Som baixo demais pode perder o efeito

2. **Use o Horário Silencioso**
   - Evite distrações em horários de descanso
   - Mantenha foco em momentos importantes

3. **Personalize os Sons**
   - Escolha sons que te motivam
   - Evite sons que causam ansiedade
